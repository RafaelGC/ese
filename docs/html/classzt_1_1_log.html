Detailed Description

Base class for logging.

You may find useful the ZELTA_LOG_WARNING(...), ZELTA_LOG_INFO(...), ZELTA_LOG_ERROR(...) and ZELTA_LOG_SUCCESS(...) directives that only logs when the ZELTALIB_DEBUG directive si defined.
